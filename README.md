# React Todo App

In this application, we can add our daily tasks that we need to do on a daily basis. It has features of removing tasks one by one or removing all tasks at once. I've learned and made this application from Thapa Technical (YouTube channel).

## Overview

The React Todo App provides a simple and intuitive interface for managing daily tasks. Users can add tasks, remove them individually, or clear the entire list at once.

## Features

- **Task Management**: Add tasks to the list and remove them individually.
- **Bulk Removal**: Clear all tasks from the list at once.
- **User-Friendly Interface**: Simple and intuitive design for easy task management.

## How to Use

To use the React Todo App, follow these steps:

1. **Access the App**: Visit the live link of the application in your web browser.

2. **Add Tasks**: Enter your daily tasks into the input field and press Enter or click the "Add Task" button.

3. **Remove Tasks**: To remove a task, hover over it and click the delete icon next to it.

4. **Clear All Tasks**: To remove all tasks at once, click the "Clear All" button.
